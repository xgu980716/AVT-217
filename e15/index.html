Instruction

1. Open AI and create a 1366*768 artboard.

2. Select rectangle tools and create a rectangle of 127px(width)*156px(height) with no storke on artboard randomly.

3. Pick #FBB03B in color mixer and fill the rectangle.

4. Select ellipse tools and create a circle of 147px(width)*147px(height) with no stroke above the rectangle. The edge of circle contacts the top edge of the rectangle.

5. Pick #FF0000 in color mixer and fill the circle.

6. Select both rectangle and circle which we have made.

7. Find align tools and change modle as "Align to Key Object".

8. Set rectangle as the key object.

9. Choose "horizontal Align Center".

10. Select star tool.

11. Create a star of 104px(width)*99px(height) with no stroke inside the circle.

12. Pick #FFFF00 in color mixer and fill the star.

13. Select both star and circle

14. Find align tools and change modle as "Align to Key Object".

15. Set circle as the key object.

16. Choose "Horizontal Align Center" and "Vertical Align Center".

17. Done
